ШАГ 1: ПОДГОТОВКА (локально)
─────────────────────────────
  python create_wallet.py
  # Сохраните ключи!
  
  @BotFather → /newbot → получите BOT_TOKEN
  @userinfobot → /start → получите ADMIN_IDS

ШАГ 2: GITHUB
─────────────
  git init
  git add .
  git commit -m "Deploy to Railway"
  git remote add origin https://github.com/ваш_username/bot2.git
  git push -u origin main

ШАГ 3: RAILWAY.APP
──────────────────
  1. railway.app → Sign up (через GitHub)
  2. New Project → Deploy from GitHub repo
  3. Выберите репозиторий bot2
  4. Railway автоматически соберет проект

ШАГ 4: БАЗА ДАННЫХ
───────────────────
  В проекте: New → Database → Add PostgreSQL
  DATABASE_URL создастся автоматически!

ШАГ 5: ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ
────────────────────────────
  Settings → Variables → Raw Editor → вставьте:
  
  BOT_TOKEN=ваш_токен
  ADMIN_IDS=ваш_id
  SOLANA_RPC_URL=https://api.devnet.solana.com
  MASTER_WALLET_PUBLIC_KEY=ваш_ключ
  MASTER_WALLET_PRIVATE_KEY=ваш_ключ
  MIN_DEPOSIT_SOL=0.01
  IMAGE_PRICE_SOL=0.05
  WITHDRAWAL_FEE_PERCENT=2

ШАГ 6: WORKER (монитор транзакций)
───────────────────────────────────
  New → Empty Service → выберите тот же репо
  Start Command: python monitor_transactions.py
  Variables → скопируйте все из основного сервиса

ШАГ 7: ИНИЦИАЛИЗАЦИЯ БД
────────────────────────
  npm i -g @railway/cli
  railway login
  railway link
  railway run python init_db.py

ШАГ 8: ПРОВЕРКА
────────────────
  Telegram → найдите бота → /start
  ✅ ГОТОВО! Бот работает 24/7!
