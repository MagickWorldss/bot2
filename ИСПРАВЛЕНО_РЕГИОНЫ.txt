❌ ПРОБЛЕМА:

  В базе данных (models.py) у таблиц Region и City 
  отсутствовали поля:
  
  • code (ISO код страны)
  • is_active (флаг активности)
  
  НО в handlers эти поля использовались!
  
  Поэтому:
  • Нельзя было добавить регион
  • Нельзя было изменить статус
  • Ошибки при попытке управления

✅ РЕШЕНИЕ:

  Добавлены недостающие поля в модели:
  
  Region:
    • name (название)
    • code (ISO код, 2 буквы, например DE)
    • is_active (активен ли регион)
    • created_at
  
  City:
    • name (название)
    • region_id (связь с регионом)
    • is_active (активен ли город)
    • created_at

   Выбрать регион → 🏙 Управление городами → ➕ Добавить город

   - Изменение статуса
   - Удаление города

ДОБАВИТЬ РЕГИОН:

  1. /god
  2. 🗂 Управление регионами
  3. ➕ Добавить регион
  4. Введите название (Germany)
  5. Введите ISO код (DE)
  6. ✅ Регион создан!


ДОБАВИТЬ ГОРОД:

  1. /god
  2. 🗂 Управление регионами
  3. Выбрать регион (Germany)
  4. 🏙 Управление городами
  5. ➕ Добавить город
  6. Введите название (Berlin)
  7. ✅ Город создан!


ИЗМЕНИТЬ СТАТУС:

  1. Выбрать регион/город
  2. Нажать 🔴 Деактивировать или 🟢 Активировать
  3. ✅ Статус изменен!


УДАЛИТЬ:

  1. Выбрать регион/город
  2. Нажать 🗑 Удалить
  3. ⚠️ Проверка на товары (если есть - нельзя удалить)
  4. ✅ Удалено!
