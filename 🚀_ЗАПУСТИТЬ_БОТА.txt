┌────────────────────────────────────────────────────────────────────────┐
│  ШАГ 1: ДОБАВИТЬ ПЕРЕМЕННЫЕ В RAILWAY (ОБЯЗАТЕЛЬНО!)                  │
└────────────────────────────────────────────────────────────────────────┘

  БЕЗ ЭТОГО БОТ НЕ ЗАПУСТИТСЯ!
  
  1️⃣  Откройте: https://railway.app/dashboard
  
  2️⃣  Войдите в ваш проект (bot2)
  
  3️⃣  Нажмите на блок бота (квадрат)
  
  4️⃣  Вкладка "Variables" (вверху)
  
  5️⃣  Нажмите "+ New Variable"
  
  6️⃣  Добавьте каждую переменную:
  
      ┌────────────────────────────────────────────────┐
      │ Variable name:  BOT_TOKEN                      │
      │ Value:  (от @BotFather)                        │
      └────────────────────────────────────────────────┘
      
      ┌────────────────────────────────────────────────┐
      │ Variable name:  ADMIN_IDS                      │
      │ Value:  (ваш ID от @userinfobot)               │
      └────────────────────────────────────────────────┘
      
      ┌────────────────────────────────────────────────┐
      │ Variable name:  MASTER_WALLET_PUBLIC_KEY       │
      │ Value:  (SOL публичный адрес)                  │
      └────────────────────────────────────────────────┘
      
      ┌────────────────────────────────────────────────┐
      │ Variable name:  MASTER_WALLET_PRIVATE_KEY      │
      │ Value:  (SOL приватный ключ)                   │
      └────────────────────────────────────────────────┘
      
      ┌────────────────────────────────────────────────┐
      │ Variable name:  DATABASE_URL                   │
      │ Value:  ${{Postgres.DATABASE_URL}}             │
      └────────────────────────────────────────────────┘
  
  7️⃣  Добавьте PostgreSQL:
      
      В проекте: + New → Database → Add PostgreSQL
      Railway создаст базу автоматически!
