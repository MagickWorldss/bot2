ПРОБЛЕМА 1: update_railway.bat не запускается
───────────────────────────────────────────────

  Решение:
  • Win + R → cmd → Enter
  • cd C:\dev\tg\bot2
  • git add .
  • git commit -m "Update"
  • git push


ПРОБЛЕМА 2: Git push не работает
─────────────────────────────────

  Решение:
  • Проверьте интернет
  • Попробуйте еще раз
  • Или используйте GitHub Desktop (графический интерфейс)


ПРОБЛЕМА 3: Railway не обновляется
───────────────────────────────────

  Решение:
  • Railway Dashboard → ваш проект
  • Settings → кнопка "Restart"
  • Или попробуйте "Redeploy"


ПРОБЛЕМА 4: Бот работает, но функции нет
──────────────────────────────────────────

  Решение:
  • Проверьте логи: Railway → View Logs
  • Убедитесь что новый деплой успешен (зеленый)
  • Подождите еще 1-2 минуты
