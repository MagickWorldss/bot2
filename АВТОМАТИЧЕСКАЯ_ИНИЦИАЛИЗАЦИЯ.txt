ВАРИАНТ 1: Обновить код на Railway (рекомендуется)
────────────────────────────────────────────────────

  1. Win + R → cmd → Enter
  
  2. В командной строке:
  
     cd C:\dev\tg\bot2
     
     git add .
     
     git commit -m "Auto init database"
     
     git push
  
  3. Railway автоматически задеплоит обновление
  
  4. Подождите 2-3 минуты
  
  5. База инициализируется АВТОМАТИЧЕСКИ!
  
  6. Telegram → @astra_shop_bot → /start
  
  ✅ ГОТОВО!
