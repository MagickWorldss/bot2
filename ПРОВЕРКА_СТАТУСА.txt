ВОПРОС 1: Добавили ли вы переменные в Railway?
───────────────────────────────────────────────

  📍 Откройте: https://railway.app/dashboard
  
  📍 Ваш проект → блок бота → Variables
  
  📍 Там должны быть:
     ✅ BOT_TOKEN
     ✅ ADMIN_IDS
     ✅ MASTER_WALLET_PUBLIC_KEY
     ✅ MASTER_WALLET_PRIVATE_KEY
     ✅ DATABASE_URL
  
  ❓ ЭТО СДЕЛАНО?
     [ ] ДА - переменные добавлены
     [ ] НЕТ - еще не добавлял


ВОПРОС 2: Добавили ли вы PostgreSQL?
──────────────────────────────────────

  📍 Railway → ваш проект
  
  📍 Должно быть 2 блока:
     ┌────────┐  ┌──────────┐
     │  Bot   │  │Postgres  │
     └────────┘  └──────────┘
  
  ❓ ЭТО СДЕЛАНО?
     [ ] ДА - PostgreSQL добавлен
     [ ] НЕТ - есть только блок бота


ВОПРОС 3: Запустили ли update_railway.bat?
────────────────────────────────────────────

  ❓ ЭТО СДЕЛАНО?
     [ ] ДА - код обновлен на GitHub
     [ ] НЕТ - еще не запускал


ВОПРОС 4: Какой статус бота в Railway?
────────────────────────────────────────

  📍 Railway → блок бота
  
  📍 Статус:
     [ ] 🟢 Running (зеленый кружок) - работает
     [ ] 🔴 Crashed (красный) - упал
     [ ] 🟡 Building (желтый) - собирается
     [ ] ⚪ Stopped (серый) - остановлен
